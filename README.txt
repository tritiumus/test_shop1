Для запуска необходимо создать БД MySQL и импортировать из файла /db/test_shop1.sql
Поменять настройки подключения к БД в файле /application/config/database.php
Поменять настройки подключения к smtp серверу в файле /application/helper/mailsender_helper.php
Поменять email адрес, для отправки уведомления оператору о поступившем заказе в файле /application/config/test_shop1.php
Изменить значение config['base_url'] на своё в файле /application/config/config.php
